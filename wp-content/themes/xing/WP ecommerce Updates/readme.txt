Updates for WP e-commerce Plugin
--------------------------------

Copy the files of this folder and paste inside 'xing/' theme folder as:

xing/wpec-custom.css
xing/wpsc-products_page.php
xing/wpsc-transaction_results.php
xing/wpsc-user-log.php

These files are required to show plugin pages properly according to the theme layout. For reference, the .wrap class is removed from these templates.

After adding the files, navigate to WordPress Settings > Store > Presentation > Advanced Theme Settings. Next, click on 'Flush Theme Cache' button and save settings.

Important: The theme only provides native structure support for WP e-commerce. i.e. no custom styling is provided from the theme. The plugin shall work normal as it does on twentyten theme.
