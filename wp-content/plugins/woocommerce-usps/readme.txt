=== WooCommerce US Postal Service Shipping Module ===

A shipping extension for USPS (United States). 

== Important Note ==

Please set your origin country to US and currency to USD

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

== Usage ==

1. Once activated, go to WooCommerce > Settings > Shipping Methods

2. Choose the method from the list at the top of the screen

3. Configure the shipping method

That's it!
